# Finance Tracker

A full stack personal finance tracker built with C++ and React

## Stack:
- Backend: C++, Crow (REST API), nlohmann-json
- Frontend: React (in progress)
- Storage: JSON file persistence 

## Status:
- Work in Progress

## Build:
```bash
cmake -B build -S . -DCMAKE_TOOLCHAIN_FILE="./vcpkg/scripts/buildsystems/vcpkg.cmake"
cmake --build build
```
